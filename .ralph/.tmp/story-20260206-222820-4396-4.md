Status: in_progress
Depends on: US-001

Description:
As a skill author, I want a shared context loading preamble so all skills can DRY-reference the same context loading instructions.

Acceptance Criteria:
- [ ] Create fab/.kit/skills/_context.md with the shared context loading convention from SKILLS.md
- [ ] Must instruct the agent to always load: fab/config.yaml (project config, tech stack), fab/constitution.md (project principles and constraints)

- [ ] Must include the Next Steps Convention: every skill MUST end output with a Next: line suggesting follow-up commands
- [ ] Must include the Next Steps lookup table mapping each skill to its appropriate Next line
- [ ] Example: a skill file would start with 'Read and follow the instructions in fab/.kit/skills/_context.md before proceeding'
- [ ] Negative case: _context.md must not contain skill-specific behavior — only shared conventions
