1. Confirm `fab/.kit/VERSION` reads `0.38.0` or higher (kit has been upgraded)
2. Check if `fab/pipelines/` directory exists — if not, skip with a note that it has already been removed

## Changes

If `fab/pipelines/` exists:

2. Print a note:
   ```
   Removed fab/pipelines/ — the pipeline orchestrator has been removed from fab-kit.
   ```

If `.gitignore` contains the line `fab/pipelines/.series-*.yaml`:

1. Remove the `fab/pipelines/.series-*.yaml` line from `.gitignore`
2. Print a note:
   ```
   Removed fab/pipelines/.series-*.yaml entry from .gitignore.
   ```
